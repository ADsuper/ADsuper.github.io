查看 sdk 安装目录和 keystore 的 md5 SHA1 等信息

发表于 2017-08-07 | 分类于 小技巧 | | 阅读次数

1. 查看 jdk 安装目录

打开命令行 输入 java -verbose ，最后两行显示的就是 安装路径

2. 查看 keystore 的 md5 SHA1 等相关信息

在 jdk bin 目录打开命令行，输入命令：keytool -list -v -keystore D:\Android\keystore\xxx.keystore

D:\Android\keystore\xxx.keystore 为 keystore 的文件路径，然后输入 keystore 密码
